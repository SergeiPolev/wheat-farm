# Wheat Farm -- Game Design Document

**Date:** 2026-03-04
**Status:** Approved
**Platform:** Steam (PC)
**Engine:** Unity 6 (6000.0.48f1), URP

---

## 1. Vision

Farm creative sandbox. Игрок строит ферму, где поле -- пространство для творчества. Выращивай культуры, деревья, кусты, меняй им цвет, ставь дома внутри поля. Красота требует усилий: семена, красители, здания стоят денег. Зарабатывай фермерством -- трать на эстетику.

**Vibe:** Spilled, Stardew Valley. Медитативное состояние с опциональными целями. Задания не давят -- предлагают.

**Core Fantasy:** "Моя ферма -- моё произведение искусства, и я его вырастил."

---

## 2. Core Loop

```
Утро: подошёл к полю -> полил культуры (подросли)
      -> проверил зрелые -> собрал урожай
      -> отнёс в амбар или продал

Деньги: купил семена / красители / здания

Производство: загрузил пшеницу в мельницу -> мука (x3 цена)

Творчество: посадил новую секцию поля
            покрасил ряд пшеницы в красный
            поставил фонарь у тропинки

Вечер: посмотрел на ферму при закате
       проверил доску контрактов
```

### Три валюты усилий

- **Время** -- растения растут не мгновенно. Полив ускоряет, удобрение ещё больше.
- **Деньги** -- от продажи урожая. Тратятся на семена, красители, здания, расширение.
- **Инвентарь** -- семена, красители, удобрения занимают место. Амбар ограничен.

### Подход: "Ферма кормит творчество"

Экономика линейная: урожай -> деньги -> покупки. Творческие инструменты (редкие растения, красители, декор) -- дорогие товары. Чтобы сделать ферму красивой, нужно сначала сделать её прибыльной.

## 3. Economy & Resources

### Единственная валюта -- монеты

### Источники дохода

| Источник | Доход | Роль |
|----------|-------|------|
| Свободная продажа урожая | Базовый | Стабильный поток, всегда доступен |
| Контракты | x1.5-3 от базовой цены | Бонус за целенаправленное выращивание |
| Редкие культуры | Высокая цена за единицу | Мотивация разблокировать новое |

### Расходы

| Расход | Стоимость | Роль |
|--------|-----------|------|
| Базовые семена (пшеница, трава) | Дёшево | Всегда доступны, хлеб насущный |
| Продвинутые семена (розы, вишня) | Средне | Разблокируются через контракты |
| Красители | Средне, расходник | Творческий ресурс |
| Удобрения | Дёшево | Ускоряют рост, QoL |
| Здания функциональные | Дорого, разово | Прогрессия |
| Декор | Средне, разово | Чистое творчество |
| Расширение территории | Очень дорого | Долгосрочная цель |

### Ключевой баланс

Базовые семена почти бесплатные -- игрок никогда не "застревает". Красители и декор -- то, на что копишь. Расширение -- долгосрочная мотивация.

### Контракты -- опциональны

Доска объявлений обновляется. Нет штрафа за невыполнение. Взял контракт -- делаешь в своём темпе. Не взял -- продаёшь свободно. Контракт не пропадает пока сам не откажешься.

Контракты могут просить переработанные товары (букет из красных роз -- дорогой контракт). Главные награды: редкие семена, рецепты красителей, чертежи зданий.

## 4. Production

Простые цепочки, 1-2 шага. Не фабрика, а сарайчик у поля.

```
Пшеница ──> Мельница ──> Мука         (x3 к цене)
Томаты  ──> Кухня    ──> Соус         (x3)
Цветы   ──> Мастерская -> Букет       (x4)
Ягоды   ──> Кухня    ──> Варенье      (x3)
Дерево  ──> Пилорама ──> Доски        (для строительства)
```

### Механика

- Положил сырьё в здание -> подождал (реальное время, минуты) -> забрал продукт
- Одно здание = один слот переработки
- Улучшение здания = больше слотов (1 -> 2 -> 3)

### Роль

- Создаёт причину возвращаться ("мука готова")
- Контракты на переработанные товары выгоднее
- Доски от пилорамы нужны для строительства -- замыкает цикл
- Фоновый процесс, не главный геймплей

## 5. Plant System

### Три категории

| Категория | Размер на grid | Примеры | Рост | Урожай |
|-----------|---------------|---------|------|--------|
| **Культуры** | 1 ячейка | Пшеница, кукуруза, томаты, морковь, подсолнух | Быстрый (1-3 мин) | Одноразовый: собрал -- ячейка пустая |
| **Кусты** | 1 ячейка, визуально крупнее | Розы, ягоды, лаванда, сирень | Средний (3-5 мин) | Многоразовый: даёт урожай, потом снова растёт |
| **Деревья** | 3x3 ячейки (ствол), крона поверх | Вишня, яблоня, дуб, ель, берёза | Долгий (10-15 мин) | Многоразовый, долгосрочная инвестиция |

### Grid Structure

Двухуровневый grid:
- **Чанки** -- единица для зданий (snap к сетке)
- **Саб-ячейки** -- единица для рендеринга (каждый стебель = GPU instance)

Культуры, кусты, деревья размещаются **свободно** (не привязаны к чанковой сетке). Здания -- строго по чанкам.

Деревья: ствол вытесняет саб-ячейки под собой. Крона рендерится поверх, НЕ вытесняет -- культуры растут под ветками.

```
Крона дерева    [листья]      <- рендерится поверх, не влияет
Культуры        ||||  ||||    <- растут под кроной
Ствол              ||         <- вытесняет саб-ячейки только здесь
```

### Стадии роста

```
Семечко (0%) -> Росток (25%) -> Рост (50%) -> Зрелость (75%) -> Урожай (100%)
```

Каждая стадия визуально отличается: масштаб растёт, цвет от бледного к насыщенному. Шейдер читает `cropState.y` -- привязка к scale и color intensity.

### Полив и удобрение

- Без полива -- растение стоит на месте, не растёт, **не умирает**
- Полив -- нормальная скорость роста
- Удобрение -- ускоренный рост (x2-x3), расходник
- Спокойная игра -- растение терпеливо ждёт

### Красители

- Применяется на любое растение в любой стадии
- Меняет тинт модели (шейдер `props.color`)
- Один краситель = одно применение на одну ячейку
- Базовые цвета (красный, синий, жёлтый) дёшевые
- Смешанные (пурпурный, бирюзовый) дороже или крафт: красный + синий = пурпурный

### Разблокировка

- Старт: пшеница, трава, морковь, роза, яблоня (5 видов)
- Контракты открывают новые виды
- Дикие растения на новых территориях (собрать семена = разблокировать)
- Цель: 20+ видов к эндгейму

### Растения не влияют друг на друга

Нет companion planting. Каждое растёт независимо. Стратегия -- в экономике, не в расстановке.

## 6. Building System

### Функциональные здания

| Здание | Функция | Разблокировка |
|--------|---------|---------------|
| **Амбар** | Хранение урожая и ресурсов. Улучшается -> больше слотов | Стартовое |
| **Колодец** | Заполняет лейку. Без колодца -- к реке на краю карты | Стартовое |
| **Магазин** | Продажа. Положил -> получил монеты | Стартовое |
| **Доска контрактов** | Показывает доступные заказы | Стартовое |
| **Мельница** | Зерно -> мука | Покупка |
| **Кухня** | Овощи/ягоды -> соусы, варенье | Покупка |
| **Мастерская** | Цветы -> букеты. Смешивание красителей | Покупка |
| **Пилорама** | Древесина -> доски | Покупка |

### Декор

Заборы, тропинки, фонари, скамейки, мостики, пруд, камни, цветочные горшки. Покупаются за монеты или крафтятся из досок. Занимают ячейки на grid.

### Размещение

- Здания привязаны к чанковой сетке (snap). Например мельница 2x2 чанка, амбар 3x2.
- Ячейки под зданием очищаются от растений.
- Здание можно перенести в любой момент.
- Декор занимает 1 ячейку.

### Улучшения

- 3 уровня на каждое функциональное здание
- Больше слотов переработки + визуально меняется модель
- Стоит монеты + доски

## 7. Tools & Controls

### Инструменты

| Инструмент | Действие | Режим | Ресурс |
|------------|----------|-------|--------|
| **Лейка** | Полив, ускоряет рост | Кисть с радиусом | Вода (наполнять у колодца) |
| **Посадка** | Сажает культуру | Кисть с радиусом | Семена из инвентаря |
| **Удобрение** | Ускоряет рост x2-3 | Кисть с радиусом | Удобрение (расходник) |
| **Краситель** | Меняет цвет растений | Кисть с радиусом | Краситель (расходник) |
| **Серп** | Собирает урожай | Кисть с радиусом | Бесплатно |
| **Корчевание** | Удаляет растение | Точечный клик | Бесплатно |
| **Строительство** | Размещает здания/декор | Grid-snap | Монеты/доски |
| **Посадка дерева/куста** | Ставит крупное растение | Свободная точка | Саженец из инвентаря |

### Управление

- **WASD** -- движение персонажа
- **Мышь** -- курсор инструмента (ограничен радиусом от персонажа)
- **ЛКМ зажать** -- применить инструмент
- **Колесо мыши** -- размер кисти
- **1-8 или Q** -- переключение инструмента (hotbar)
- **Tab** -- инвентарь
- **E** -- взаимодействие со зданиями
- **Средняя кнопка** -- вращение камеры
- **Ctrl + колесо** -- зум

### Продвинутые инструменты (поздний контент)

- Автолейка
- Дроны/роботы-сборщики
- Автотрактор
- Серп-меч (махать и рубить траву)
- Бег, рывок

## 8. Progression

### Три оси

**1. Территория (горизонтальная)**

```
Старт: ~20x20 чанков
  -> Расширения по сторонам (+20x20 каждое, дорожают)
  -> Новые территории: дикие растения, камни, водоёмы
```

**2. Каталог растений (вертикальная)**

```
Старт: 5 видов (пшеница, трава, морковь, роза, яблоня)
  -> Контракты открывают новые виды
  -> Дикие растения на новых территориях
  -> Цель: 20+ видов
```

Новые виды -- главная награда контрактов. Расширяет палитру для творчества.

**3. Здания и инструменты (функциональная)**

```
Старт: амбар, колодец, магазин, доска контрактов
  -> Покупка: мельница, кухня, мастерская, пилорама
  -> Улучшения: 3 уровня каждого здания
  -> Поздний контент: автолейки, дроны, трактор
```

### Темп

- 1 час: освоился, купил мельницу, 2-3 новых растения
- 5 часов: половина каталога, 2-3 расширения, производство работает
- 10 часов: большая ферма, декорирование, контракты на редкие букеты

Нет потолка: территория расширяется, комбинаторика (вид x цвет x расположение) бесконечна.

## 9. Day/Night Cycle

Цель: эстетика, не ограничение.

### Цикл

```
Рассвет (2 мин) -> День (6 мин) -> Закат (2 мин) -> Ночь (4 мин)
                         ~14 минут полный цикл
```

### Что меняется

- **Освещение** -- тёплый рассвет, нейтральный день, оранжевый закат, синяя ночь
- **Фонари** -- включаются ночью автоматически
- **Растения** -- чуть темнее ночью, красители играют иначе при разном свете
- **Рост продолжается** -- ночь не останавливает ничего

### Чего нет

- Нет "успей до конца дня"
- Нет ограничения действий ночью
- Нет сна/усталости
- Нет NPC с расписанием

### Управление временем

- Ускорение (x2, x4) -- быстрее дождаться урожая
- Пауза времени -- зафиксировать закат пока декорируешь

## 10. Camera & Visual

### Камера

- Изометрическая, ортографическая (Cinemachine)
- **Зум** -- колесо мыши. От стеблей до обзора всей фермы
- **Вращение** -- 4 фиксированных угла по 90 градусов, snap с плавным переходом
- **Следование** -- за персонажем с мягким damping
- **Свободный режим** -- средняя кнопка отводит камеру для обзора

### Визуальный стиль

- Low-poly модели с приятными пропорциями
- Яркие, насыщенные цвета
- Мягкие тени, нет жёстких контрастов
- Ветер шевелит культуры и кроны (vertex animation в шейдере)
- Партиклы: пыльца при поливе, листья при срезке, искры при крафте

### Post-processing (URP Volume)

- Bloom мягкий (свечение фонарей ночью)
- Color grading (теплее днём, холоднее ночью)
- Vignette лёгкая
- Ambient Occlusion

## 11. Technical Architecture

### Stack

| Технология | Роль |
|-----------|------|
| **VContainer** | DI с иерархией скоупов (вместо AllServices) |
| **R3** | Reactive programming (ReactiveProperty, Observable) |
| **UniTask** | async/await (вместо корутин) |
| **MVP** | UI паттерн (View / Presenter / Service) |
| **Feature Installers** | Модульная регистрация систем |
| **Assembly Definitions** | Модульная компиляция |
| **GPU Instanced Indirect** | Рендеринг культур (существующий) |

### Scope Hierarchy

```
RootScope (DontDestroyOnLoad)
├── SaveService, TimeService, ConfigDatabase
│
└── GameScope (игровая сессия)
    ├── WalletService, InventoryService, ShopService, ContractService
    ├── DayNightService, InputService, CameraService
    │
    └── FarmScope (жизнь фермы)
        ├── ChunkSystem, PlantSystem, BrushService
        ├── BuildingService, ProductionService
        ├── TreePlacementService
        └── CropRenderer (per-chunk)
```

### Feature Installers

```
Features/
  Farming/
    FarmingInstaller.cs     -- PlantSystem, BrushService, ChunkSystem
  Buildings/
    BuildingsInstaller.cs   -- BuildingService, ProductionService
  Economy/
    EconomyInstaller.cs     -- WalletService, ShopService, ContractService
  DayNight/
    DayNightInstaller.cs    -- DayNightService, LightingController
  Inventory/
    InventoryInstaller.cs   -- InventoryService
```

### Data Flow (R3)

```
PlantSystem:
  ReactiveProperty<float> GrowthProgress
  ReactiveEvent<HarvestData> OnHarvested

WalletService:
  ReactiveProperty<int> Coins
  ReactiveEvent<int> OnCoinsChanged

InventoryService:
  ObservableList<InventoryItem> Items
  ReactiveEvent<InventoryItem> OnItemAdded
```

UI подписывается через Presenter:
```
WalletPresenter(IWalletService wallet, WalletView view)
  → wallet.Coins.Subscribe(view.UpdateCoinsText)
```

### MVP для UI

```
UI/
  HUD/
    HUDView.cs         -- MonoBehaviour, только визуал
    HUDPresenter.cs    -- чистый C#, IInitializable, подписки
  Shop/
    ShopView.cs
    ShopPresenter.cs
  Inventory/
    InventoryView.cs
    InventoryPresenter.cs
```

### Assembly Structure

```
WheatFarm.Core            -- интерфейсы, базовые типы, extensions
WheatFarm.Core.Editor     -- editor tools
WheatFarm.Infrastructure  -- VContainer scopes, bootstrap, save/load
WheatFarm.Farming         -- PlantSystem, ChunkSystem, CropRenderer, BrushService
WheatFarm.Buildings       -- BuildingService, ProductionService
WheatFarm.Economy         -- Wallet, Shop, Contracts, Inventory
WheatFarm.UI              -- Views, Presenters
WheatFarm.DayNight        -- время суток, освещение
WheatFarm.Player          -- движение, инструменты, ToolService
```

### Рефакторинг существующего кода

| Текущее | Новое |
|---------|-------|
| `AllServices` (синглтон ServiceLocator) | VContainer scopes |
| `GameStateMachine` / `IState` / `ITick` | VContainer `IInitializable` + `ITickable` |
| `CropFieldData` (один плоский grid) | `ChunkSystem` (chunk-based grid) + per-chunk `CropRenderer` |
| `FieldToolsService` (всё в одном Tick) | Отдельные `ITool` реализации + `BrushService` |
| `Tool.cs` / `ToolHandler.cs` (3 класса в файле) | Tool per file + `ToolService` |
| `WalletService` (две валюты) | `WalletService` (монеты) + `InventoryService` (предметы) |
| `CropsType` enum (4 типа) | `PlantData` ScriptableObject per вид (data-driven) |
| `WindowBase` (логика в View) | MVP: View + Presenter |
| Корутины | UniTask async/await |
| Прямые вызовы / C# events | R3 ReactiveProperty / ReactiveEvent |

### Новые системы

| Система | Ответственность |
|---------|----------------|
| `ChunkSystem` | Grid чанков, расширение территории, snap зданий |
| `PlantSystem` | Рост по времени, стадии, модификаторы скорости |
| `InventoryService` | Семена, красители, удобрения, урожай, слоты, лимит амбара |
| `BuildingService` | Размещение, перемещение, удаление, уровни |
| `ProductionService` | Очереди переработки, таймеры, рецепты |
| `ContractService` | Генерация контрактов, прогресс, награды |
| `DayNightService` | Цикл времени, управление освещением, ускорение/пауза |
| `BrushService` | Радиус кисти, применение инструмента к области |
| `TreePlacementService` | Свободное размещение, вытеснение саб-ячеек под стволом |
| `ShopService` | Каталог, покупка, разблокировка по прогрессу |

### Input -> Render Data Flow

```
Input -> BrushService (определяет область)
  -> текущий ITool (что делаем)
  -> ChunkSystem (находит затронутые чанки)
  -> PlantSystem / BuildingService (применяет изменение)
  -> CropRenderer (обновляет ComputeBuffer)
```
